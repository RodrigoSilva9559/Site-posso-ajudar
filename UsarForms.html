Utilizando Microsoft Forms em Sala de Aula: Um Guia Passo a Passo

    O Microsoft Forms é uma ferramenta versátil e intuitiva para a criação de formulários e questionários interativos. Suas funcionalidades abrangentes e integração com outros serviços do Microsoft 365 tornam-no uma escolha poderosa para educadores e administradores que buscam coletar e analisar dados de forma eficiente.



Acesse o Microsoft Forms

Abra o navegador e vá para o site: entre aqui

Faça seu login, aqui explica como: veja como

Clique em "Forms" no menu de aplicativos.

Clique em "Novo Formulário" ou "Novo Questionário" para começar

  • Novo Formulário: Ideal para pesquisas e coleta de informações
  • Novo Questionário: Perfeito para testes e avaliações,
    pois permite adicionar pontos para respostas.

    • Dê um Título e Descrição ao Formulário



  • Escolha Múltipla: Ideal para perguntas de múltipla escolha.
  • Texto: Para respostas abertas.
  • Classificação: Para perguntas em que os alunos classificam algo
  • Data: Para selecionar datas.

    Compartilhe o Formulário

  • Clique em "Coletar Respostas" para obter o link do formulário.
  • Você pode copiar o link e enviar por e-mail, inserir em um site, ou compartilhar diretamente pelo Teams.
  • Também é possível gerar um QR code ou enviar por e-mail diretamente da plataforma.
