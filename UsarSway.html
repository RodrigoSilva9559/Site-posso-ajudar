Utilizando Microsoft Sway em Sala de Aula: Um Guia Passo a Passo

    O Microsoft Sway é uma ferramenta inovadora e fácil de usar para a criação de conteúdo multimídia interativo e apresentações dinâmicas. Suas funcionalidades e integração com outros serviços da Microsoft tornam-no uma excelente escolha para professores e estudantes que buscam criar materiais educativos e projetos de alta qualidade.

Clique em "Criar novo" para iniciar uma nova apresentação.

No topo da página, adicione um título relevante para a apresentação que você está criando.

Clique em "Adicionar" para ver as opções de cartões.

  • Texto: Adicione títulos, parágrafos e citações.
  • Multimídia: Inclua imagens, vídeos, áudios e incorporações.
  • Gráficos: Adicione gráficos, tabelas e outros elementos visuais.

    • Design: Clique em "Design" para escolher um layout para sua apresentação.

    Experimente diferentes layouts para ver qual fica melhor com seu conteúdo.

    Clique em "Reproduzir" para visualizar como sua apresentação será exibida para os outros.
